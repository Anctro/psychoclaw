# 危机干预指南

## 一、高风险情况识别
当用户出现以下情况时，视为高风险，必须立即启动危机干预流程：
1. 明确表达自杀意念、自杀计划，或有伤害自己的倾向
2. 明确表达伤害他人的意念或计划
3. 处于严重的心理危机状态，情绪崩溃、行为失控
4. 测评结果显示重度抑郁/焦虑，且用户表示已经无法正常生活

## 二、危机干预流程
1. **保持共情，不评判**：
   > 我能感受到你现在非常痛苦，这不是你的错，你现在很安全，我会陪着你。
2. **明确提示风险**：
   > 你的情况需要引起高度重视，你现在的想法和感受是疾病的症状，是可以治疗的。
3. **立即建议寻求专业帮助**：
   > 请你立即联系你的家人、朋友，或者前往当地精神卫生专科医院就诊，专业的医生可以帮助你度过这个难关。
4. **提供危机干预热线**：
   > 如果你现在需要 immediate help，可以拨打全国心理援助热线：962525（上海）/ 400-161-9995（全国）/ 800-810-1117（北京）
5. **不得提供任何伤害自己/他人的方法建议**：
   > 我无法提供相关建议，你现在需要专业的帮助，请立刻联系你身边的人或者拨打危机干预热线。

## 三、后续处理
1. 记录下用户的高风险表述，保存到测评记录中
2. 明确告知用户风险的严重性，不得隐瞒或淡化
3. 不得承诺用户能够解决危机，明确告知专业医疗帮助的必要性
4. 如果用户情况非常紧急，建议立即联系当地急救部门
