# 问卷星导入格式官方规范

## 一、文本快速导入规范
### 格式要求
1. 问卷标题放在最开头，用【】包裹，比如：【心理健康测评问卷】
2. 问卷说明放在标题下面，可省略
3. 每题用数字编号开头，如「1. 题目内容」
4. 选项用大写字母开头，如「A. 选项内容」
5. 题型标注放在题目末尾，用【】包裹，比如：
   - 单选题：默认，可不标注
   - 多选题：【多选题】
   - 量表题：【量表题|1-5分】
   - 填空题：【填空题】
   - 多行填空题：【多行填空题】
   - 排序题：【排序题】
   - 评分题：【评分题|1-10分】

### 示例
```
【心理健康测评问卷】
本问卷用于评估您的心理健康状态，所有信息严格保密，请根据实际情况选择。

1. 您最近两周情绪低落的频率？
A. 完全没有
B. 偶尔
C. 经常
D. 几乎每天

2. 您最近有以下哪些症状？【多选题】
A. 失眠
B. 食欲下降
C. 注意力不集中
D. 容易烦躁

3. 您对目前生活的满意度？【量表题|1-5分】
1分 → 非常不满意
2分 → 不满意
3分 → 一般
4分 → 满意
5分 → 非常满意

4. 您的年龄是？【填空题】
```

## 二、Excel快速导入规范
### 表头要求
必须包含以下列（顺序可调整）：
| 列名 | 说明 | 必填 |
|------|------|------|
| 题目序号 | 题目的编号，从1开始递增 | 是 |
| 题目内容 | 题目的具体内容 | 是 |
| 题型 | 题型：单选题/多选题/量表题/填空题/... | 是 |
| 选项1 ~ 选项N | 选项内容，最多支持10个选项 | 单选/多选/量表题必填 |

### 题型可选值
```
单选题、多选题、判断题、填空题、多行填空题、量表题、排序题、评分题、矩阵单选题、矩阵多选题、矩阵量表题、下拉框题、滑动条题、日期题、时间题、城市题、省市区题、姓名题、手机题、邮箱题、地址题、身份证题、学历题、职业题
```

|---------|---------|------|-------|-------|-------|-------|
| 1 | 您的性别是？ | 单选题 | 男 | 女 | 保密 |  |
| 2 | 您的兴趣爱好有哪些？ | 多选题 | 阅读 | 运动 | 电影 | 旅游 |
| 3 | 您的工作年限是？ | 填空题 |  |  |  |  |

## 三、导入注意事项
1. 文本导入时，题目和选项之间不要有空行，不同题目之间用空行分隔
2. Excel导入时，不要合并单元格，所有内容放在第一个sheet
3. 量表题/评分题的选项顺序要和分值顺序一致
4. 导入前建议先用本工具的校验功能检查格式是否正确
5. 导入后建议在问卷星中预览一下，确认题型和选项是否正确
