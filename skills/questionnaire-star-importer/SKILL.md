# 问卷星快速导入工具

## 概述
本技能用于快速将任意格式的问卷内容转换为问卷星官方支持的导入格式，自动适配问卷星的文本导入和Excel导入规范，支持多种题型自动识别，无需手动调整格式，极大提升问卷制作效率。

## 核心功能
1. **全格式输入支持**：支持纯文本(.txt)、Markdown(.md)、Word(.docx)、PDF、Excel(.xlsx/.xls)、心理量表JSON等几乎所有常见格式的输入
2. **智能超限处理**：自动检测文件大小/文本长度限制，提供两种处理模式供用户选择：(1) 自动截断超限部分+优化建议；(2) 拆分多份文件分批导入，同时自动给出超限原因和优化方案
3. **多渠道输出（按优先级自动选择）**：
   - 优先级1：自动生成飞书云文档并返回分享链接，支持直接在线预览和下载
   - 优先级2：生成本地临时HTTP访问链接（有效期24小时），无需下载直接访问
   - 优先级3：直接返回完整文本内容，可直接复制粘贴使用
4. **智能输出格式适配**：
   - 单份问卷：自动生成问卷星「Word快速导入」格式文件（纯文本无格式，完全符合官方规范）
   - 多份问卷/批量导入：自动生成「Excel快速导入」格式文件
   - 支持手动指定输出格式
5. **全题型自动识别**：自动识别单选、多选、量表、矩阵、排序、填空、多项填空、表格、比重、段落说明等问卷星支持的所有10种官方题型
6. **矩阵题智能补全**：当矩阵题缺少题目内容时，自动基于行标题和评分维度生成合理的引导语，无需手动补充
7. **心理量表专属适配**：内置支持将PsychoClaw的心理量表JSON一键转换为问卷星格式，自动合并相同选项的量表题为矩阵题，大幅减少题目数量，提升作答体验
8. **格式自动校验与修复**：自动检查导入格式是否符合问卷星规范，可自动修复可识别的格式问题（如缺选项、题型标识错误、格式混乱等）
9. **批量处理**：支持一次性转换多个问卷/量表，批量生成导入文件

## 题型支持列表（完全匹配问卷星官方10种题型）
| 题型 | 标识 | 格式说明 |
|------|------|----------|
| 单选题 | （默认无需标识） | 包含题号、题目内容、选项，系统自动识别 |
| 多选题 | `[多选题]` | 包含题号、题目内容、标签[多选题]、选项 |
| 填空题 | `[填空题]` | 包含题号、题目内容，无需选项 |
| 多项填空题 | `[多项填空题]` | 包含题号、标题、内容中包含下划线符号「____」标识填空位置 |
| 矩阵题 | `[矩阵题]` | 包含序号、题目内容、标签[矩阵题]、第一行为统一选项（空格分隔）、后续行为行标题 |
| 表格题 | `[表格题]` | 包含序号、题目内容、标签[表格题]、第二行为问题选项、第三行为列标题（表头）、后续行为行内容 |
| 排序题 | `[排序题]` | 包含题号、题目内容、标签[排序题]、选项 |

| 段落说明 | `[段落说明]` | 包含文字内容、标签[段落说明]，非题目类引导内容 |
| 量表题 | `[量表题]` | 包含题号、题目内容、标签[量表题]、选项（每个选项单独一行） |

## 矩阵题特殊处理规则
⚠️ **格式强制要求**：矩阵题的统一选项必须放在题目后的第二行，所有选项在同一行用空格分隔，不允许换行，行标题从第三行开始每行一个。

当输入内容中矩阵题缺少题目内容时，系统会自动基于行标题的共性和评分维度生成合理的引导语，无需手动补充：
1. 自动识别所有行标题的共同主题（例如：均为学校安全管理相关内容）
2. 结合评分选项维度（例如：满意度评分、重要性评分）
3. 自动生成符合场景的引导语（例如：「请对以下各项学校安全管理工作的满意度作个评价」）

### 矩阵题格式示例
```
3. 请对学校的安全管理制度的满意度作个评价 [矩阵题]
很不满意 不满意 一般 满意 很满意
寝室里面用电安全管理
食堂就餐安全管理
教学楼消防安全管理
```
如果输入时缺少题目内容，会自动补全为符合上述格式的标准内容

## 使用流程
### 标准执行流程
1. **依赖检查**：自动检查相关skills和安装包依赖（包括文件解析、Office处理、飞书API等依赖），缺失自动安装
2. **内容解析**：自动解析文本、Word、Excel、PDF等格式的输入内容，统一转换为标准Markdown文本
3. **超限检测与处理**：自动检测文件大小/文本长度是否超出问卷星导入限制：
   - 若未超限：继续执行
   - 若超限：提示用户选择处理模式：(1) 自动截断超限部分+优化建议；(2) 拆分多份文件分批导入
4. **格式生成**：根据输入内容类型自动选择输出格式：
   - 单份问卷：生成符合问卷星规范的Word导入文件（纯文本无格式）
   - 多份问卷/批量导入：生成Excel批量导入文件
   - 支持手动指定输出格式
5. **心理量表专属处理**：若输入为心理量表JSON，自动合并相同选项的量表题为矩阵题，减少重复内容
6. **校验调整**：自动校验生成的格式是否符合问卷星官方规范，可自动修复可识别的格式问题（如缺选项、题型标识错误、格式混乱等）
7. **多渠道输出**：按优先级自动选择输出渠道：
   - 优先调用feishu-doc skill生成飞书云文档，返回分享链接
   - 若飞书接口不可用：生成本地临时HTTP服务（端口随机，有效期24小时），返回访问链接
   - 若本地服务启动失败：直接返回完整转换后的文本内容
8. **结果通知**：将最终输出结果和使用说明返回给用户

### 一键转换流程（推荐）
直接使用通用转换脚本，自动识别输入格式、输出格式和输出渠道：
```bash
python3 scripts/universal-converter.py <输入文件>
```
脚本会自动完成上述全流程操作，无需手动干预

### 常用参数说明
| 参数 | 说明 |
|------|------|
| `--output-format <word/excel>` | 手动指定输出格式，可选值：word（单份导入）、excel（批量导入） |
| `--output-mode <feishu/local/text>` | 手动指定输出渠道，可选值：feishu（飞书文档）、local（本地HTTP链接）、text（直接返回文本） |
| `--over-limit-mode <truncate/split>` | 手动指定超限处理模式，可选值：truncate（截断超限部分）、split（拆分多份文件） |
| `--output-path <path>` | 手动指定本地文件保存路径 |

### 示例命令
```bash
# 强制输出Word格式，优先返回飞书文档链接
python3 scripts/universal-converter.py 问卷.md --output-format word --output-mode feishu

# 强制输出Excel格式，超限自动拆分，返回本地HTTP链接
python3 scripts/universal-converter.py 批量问卷.xlsx --output-format excel --over-limit-mode split --output-mode local

# 直接返回转换后的文本内容
python3 scripts/universal-converter.py 量表.json --output-mode text
```

### 支持的输入格式
| 格式 | 说明 |
|------|------|
| .txt/.md | 纯文本/Markdown格式问卷 |
| .docx | Word文档问卷，自动提取所有文本内容 |
| .pdf | PDF格式问卷，自动OCR提取文本 |
| .xlsx/.xls | Excel格式问卷，支持批量导入模板 |
| .json | PsychoClaw心理量表JSON格式，一键转换 |

### 导入到问卷星
生成的文件无需任何调整，直接导入问卷星即可自动生成完整问卷，包含所有题目、选项、题型设置：
1. Word格式：选择「导入Word」上传即可
2. Excel格式：选择「导入Excel」上传即可
3. 支持所有题型自动识别，无需手动调整
