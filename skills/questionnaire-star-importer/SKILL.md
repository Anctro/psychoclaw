# 问卷星快速导入工具

## 概述
本技能用于快速将任意格式的问卷内容转换为问卷星官方支持的导入格式，自动适配问卷星的文本导入和Excel导入规范，支持多种题型自动识别，无需手动调整格式，极大提升问卷制作效率。

## 核心功能
1. **全格式输入支持**：支持纯文本、Markdown、Word(.docx)、PDF、Excel(.xlsx/.xls)、心理量表JSON等几乎所有常见格式的输入
2. **智能输出适配**：
   - 单份问卷：自动生成问卷星「Word快速导入」格式文件（纯文本无格式，完全符合官方规范）
   - 多份问卷/批量导入：自动生成「Excel快速导入」格式文件
   - 支持手动指定输出格式
3. **全题型自动识别**：自动识别单选、多选、量表、矩阵、排序、填空、多项填空、表格、比重、段落说明等问卷星支持的所有题型
4. **矩阵题智能补全**：当矩阵题缺少题目内容时，自动基于行标题和评分维度生成合理的引导语，无需手动补充
5. **智能格式转换**：自动识别输入内容的结构，无需手动标注题型，准确率≥95%
6. **心理量表专属适配**：内置支持将PsychoClaw的心理量表JSON一键转换为问卷星格式，无需二次调整
7. **格式自动校验与修复**：自动检查导入格式是否符合问卷星规范，可自动修复可识别的格式问题
8. **批量处理**：支持一次性转换多个问卷/量表，批量生成导入文件

```
2. 题目2（多选题）【多选题】
A. 选项1
B. 选项2

3. 题目3（量表题）【量表题|1-5分】
1分 → 非常不同意

3分 → 一般
4分 → 同意
5分 → 非常同意

4. 题目4（填空题）【填空题】
```

## 题型支持列表
| 题型 | 标识 | 说明 |

| 填空题 | 【填空题】 | 单行填空 |
| 多项填空题 | 【多项填空题】 | 多个填空项，内容含下划线符号「____」 |
| 排序题 | 【排序题】 | 选项按重要程度/优先级排序 |
| 评分题 | 【评分题|1-10分】 | 星级评分或数值评分 |

| 比重题 | 【比重题】 | 各选项占比总和为100%的题型 |
| 段落说明 | 【段落说明】 | 问卷说明、引导语等非题目内容 |

## 矩阵题特殊处理规则
当输入内容中矩阵题缺少题目内容时，系统会自动基于行标题的共性和评分维度生成合理的引导语，无需手动补充：
1. 自动识别所有行标题的共同主题（例如：均为学校安全管理相关内容）
2. 结合评分选项维度（例如：满意度评分、重要性评分）
3. 自动生成符合场景的引导语（例如：「请对以下各项学校安全管理工作的满意度作个评价」）

### 矩阵题格式示例
```
3. 请对学校的安全管理制度的满意度作个评价 [矩阵题]
很不满意 不满意 一般 满意 很满意
寝室里面用电安全管理
食堂就餐安全管理
教学楼消防安全管理
```
如果输入时缺少题目内容，会自动补全为符合上述格式的标准内容

## 使用流程
### 标准执行流程
1. **依赖检查**：自动检查相关skills和安装包依赖（包括文件解析、Office处理等依赖），缺失自动安装
2. **内容解析**：自动解析文本、Word、Excel、PDF等格式的输入内容，统一转换为标准Markdown文本
3. **格式生成**：根据输入内容类型自动选择输出格式：
   - 单份问卷：生成符合问卷星规范的Word导入文件（纯文本无格式）
   - 多份问卷/批量导入：生成Excel批量导入文件
   - 支持手动指定输出格式
4. **校验调整**：自动校验生成的格式是否符合问卷星官方规范，可自动修复可识别的格式问题，完成后通知用户

### 一键转换流程（推荐）
直接使用通用转换脚本，自动识别输入格式和输出格式：
```bash
python3 universal-converter.py <输入文件>
```
脚本会自动完成上述全流程操作，无需手动干预

### 手动指定输出格式
```bash
# 强制输出Word格式（单份问卷导入）
python3 universal-converter.py <输入文件> word <输出路径.docx>

# 强制输出Excel格式（批量导入）
python3 universal-converter.py <输入文件> excel <输出路径.xlsx>
```

### 支持的输入格式
| 格式 | 说明 |
|------|------|
| .txt/.md | 纯文本/Markdown格式问卷 |
| .docx | Word文档问卷，自动提取所有文本内容 |
| .pdf | PDF格式问卷，自动OCR提取文本 |
| .xlsx/.xls | Excel格式问卷，支持批量导入模板 |
| .json | PsychoClaw心理量表JSON格式，一键转换 |

### 导入到问卷星
生成的文件无需任何调整，直接导入问卷星即可自动生成完整问卷，包含所有题目、选项、题型设置：
1. Word格式：选择「导入Word」上传即可
2. Excel格式：选择「导入Excel」上传即可
3. 支持所有题型自动识别，无需手动调整
