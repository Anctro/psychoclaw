# 问卷星快速导入工具

## 概述
本技能用于快速将任意格式的问卷内容转换为问卷星官方支持的导入格式，自动适配问卷星的文本导入和Excel导入规范，支持多种题型自动识别，无需手动调整格式，极大提升问卷制作效率。

## 核心功能
1. **全格式输入支持**：支持纯文本、Markdown、Word(.docx)、PDF、Excel(.xlsx/.xls)、心理量表JSON等几乎所有常见格式的输入
2. **智能输出适配**：
   - 单份问卷：自动生成问卷星「Word快速导入」格式文件（纯文本无格式，完全符合官方规范）
   - 多份问卷/批量导入：自动生成「Excel快速导入」格式文件
   - 支持手动指定输出格式
3. **全题型自动识别**：自动识别单选、多选、量表、矩阵、排序、填空、多行填空等问卷星支持的所有题型
4. **智能格式转换**：自动识别输入内容的结构，无需手动标注题型，准确率≥95%
5. **心理量表专属适配**：内置支持将PsychoClaw的心理量表JSON一键转换为问卷星格式，无需二次调整
6. **格式自动校验与修复**：自动检查导入格式是否符合问卷星规范，可自动修复可识别的格式问题
7. **批量处理**：支持一次性转换多个问卷/量表，批量生成导入文件

## 问卷星导入格式规范
### 1. 文本快速导入格式（官方标准）
```
【问卷标题】
问卷说明内容（可选）

1. 题目1（单选题）
A. 选项1
B. 选项2
C. 选项3

2. 题目2（多选题）【多选题】
A. 选项1
B. 选项2
C. 选项3

3. 题目3（量表题）【量表题|1-5分】
1分 → 非常不同意
2分 → 不同意
3分 → 一般
4分 → 同意
5分 → 非常同意

4. 题目4（填空题）【填空题】
```

| 1 | 问题1 | 单选题 | 选项1 | 选项2 | 选项3 |  |  |
| 2 | 问题2 | 多选题 | 选项1 | 选项2 | 选项3 |  |  |
| 3 | 问题3 | 量表题 | 非常不同意 | 不同意 | 一般 | 同意 | 非常同意 |

## 使用流程
### 一键转换流程（推荐）
直接使用通用转换脚本，自动识别输入格式和输出格式：
```bash
python3 universal-converter.py <输入文件>
```
脚本会自动：
1. 识别输入文件格式（txt/md/docx/pdf/xlsx）
2. 自动解析内容、识别题型
3. 校验格式，修复可自动修正的问题
4. 单份问卷自动输出Word格式，多份/Excel输入自动输出Excel格式
5. 生成符合问卷星规范的导入文件

### 手动指定输出格式
```bash
# 强制输出Word格式（单份问卷导入）
python3 universal-converter.py <输入文件> word <输出路径.docx>

# 强制输出Excel格式（批量导入）
python3 universal-converter.py <输入文件> excel <输出路径.xlsx>
```

### 支持的输入格式
| 格式 | 说明 |
|------|------|
| .txt/.md | 纯文本/Markdown格式问卷 |
| .docx | Word文档问卷，自动提取所有文本内容 |
| .pdf | PDF格式问卷，自动OCR提取文本 |
| .xlsx/.xls | Excel格式问卷，支持批量导入模板 |
| .json | PsychoClaw心理量表JSON格式，一键转换 |

### 导入到问卷星
生成的文件无需任何调整，直接导入问卷星即可自动生成完整问卷，包含所有题目、选项、题型设置：
1. Word格式：选择「导入Word」上传即可
2. Excel格式：选择「导入Excel」上传即可
3. 支持所有题型自动识别，无需手动调整
