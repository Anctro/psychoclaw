# 问卷星快速导入工具

## 概述
本技能用于快速将任意格式的问卷内容转换为问卷星官方支持的导入格式，自动适配问卷星的文本导入和Excel导入规范，支持多种题型自动识别，无需手动调整格式，极大提升问卷制作效率。

## 核心功能
1. **多格式输入支持**：支持纯文本、Markdown、心理量表JSON、现有问卷文档等多种输入格式
2. **双格式输出**：同时支持生成问卷星「文本快速导入」格式和「Excel快速导入」格式
3. **题型自动识别**：自动识别单选题、多选题、量表题、填空题、开放题等常见题型
4. **心理量表适配**：内置支持将PsychoClaw的心理量表JSON直接转换为问卷星格式
5. **格式自动校验**：自动检查导入格式是否符合问卷星规范，不符合时给出修正建议
6. **批量处理**：支持一次性转换多个问卷/量表，批量生成导入文件

## 问卷星导入格式规范
### 1. 文本快速导入格式（官方标准）
```
【问卷标题】
问卷说明内容（可选）

1. 题目1（单选题）
A. 选项1
B. 选项2
C. 选项3

2. 题目2（多选题）【多选题】
A. 选项1
B. 选项2
C. 选项3

3. 题目3（量表题）【量表题|1-5分】
1分 → 非常不同意
2分 → 不同意
3分 → 一般
4分 → 同意
5分 → 非常同意

4. 题目4（填空题）【填空题】
```

### 2. Excel快速导入格式（官方标准）
| 题目序号 | 题目内容 | 题型 | 选项1 | 选项2 | 选项3 | 选项4 | 选项5 |
|---------|---------|------|-------|-------|-------|-------|-------|
| 1 | 问题1 | 单选题 | 选项1 | 选项2 | 选项3 |  |  |
| 2 | 问题2 | 多选题 | 选项1 | 选项2 | 选项3 |  |  |
| 3 | 问题3 | 量表题 | 非常不同意 | 不同意 | 一般 | 同意 | 非常同意 |
| 4 | 问题4 | 填空题 |  |  |  |  |  |

## 题型支持列表
| 题型 | 标识 | 说明 |
|------|------|------|
| 单选题 | 【单选题】/ 默认 | 单选，默认题型 |
| 多选题 | 【多选题】 | 多选，可选择多个选项 |
| 量表题 | 【量表题|1-N分】 | 评分量表，支持自定义分值范围 |
| 填空题 | 【填空题】 | 单行填空 |
| 多行填空题 | 【多行填空题】 | 多行文本输入 |
| 排序题 | 【排序题】 | 选项排序 |
| 评分题 | 【评分题|1-10分】 | 星级评分或数值评分 |

## 使用流程
### 步骤1：输入问卷内容
支持以下输入方式：
1. 直接粘贴纯文本/Markdown格式的问卷内容
2. 传入PsychoClaw心理量表JSON文件路径
3. 上传现有问卷文档（.txt/.md/.json）

### 步骤2：选择输出格式
- 生成文本导入格式：直接复制到问卷星「文本快速导入」框即可导入
- 生成Excel导入格式：下载Excel文件后直接上传到问卷星「Excel快速导入」即可

### 步骤3：导入到问卷星
生成的文件无需任何调整，直接导入问卷星即可自动生成完整问卷，包含所有题目、选项、题型设置。
