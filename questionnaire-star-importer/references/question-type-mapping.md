# 题型映射规则说明

## 自动识别规则
脚本会自动根据题目内容识别题型，无需手动标注：

### 1. 单选题（默认）
- 没有标注任何题型的题目默认识别为单选题
- 选项数量≤5个，且没有「可多选」「多选」字样的题目识别为单选题

### 2. 多选题
- 题目中包含「可多选」「多选」「请选择所有符合的」字样
- 选项数量≥6个的题目默认识别为多选题
- 手动标注【多选题】强制识别为多选题

### 3. 量表题/评分题
- 题目中包含「请打分」「满意度」「同意程度」「符合程度」字样
- 选项为数字评分（1-5分、1-10分等）
- 手动标注【量表题|1-5分】【评分题|1-10分】强制识别为对应分值的量表题

### 4. 填空题
- 题目中包含「请填写」「您的XX是？」「请输入」字样
- 没有选项的题目默认识别为填空题
- 手动标注【填空题】【多行填空题】强制识别为对应类型

### 5. 排序题
- 题目中包含「请排序」「按优先级排序」「排序」字样
- 手动标注【排序题】强制识别为排序题

## 自定义题型映射
你可以修改脚本中的题型识别规则，添加自定义的题型关键词：

### 示例：添加自定义题型识别
在`parse_question`函数中添加：
```python
# 自定义：题目包含「是否」识别为判断题
if "是否" in q_content:
    q_type = "判断题"
```

### 示例：添加新题型支持
在输出格式模板中添加新题型的支持：
```python
# 支持判断题
if q_type == "判断题":
    options = ["对", "错"]
```
